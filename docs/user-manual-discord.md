**Daily Use**
_Core, high-frequency commands._

- **!ft add|append <list>** - save/append trading list  
  Example: `!ft append GS M5, GD 5`
- **!lf add|append <list>** - save/append looking-for list  
  Example: `!lf append GDratini 5`
- **!id add <number>** - save your TPPC ID  
  Example: `!id add 123456`
- **!rarity <pokemon>** - quick rarity lookup  
  Example: `!rarity Dratini`
- **!calc <mode> ...** - level/exp/money calc. `!calc help` to get started.  
  Example: `!calc l2e 100`
- **!lb <board>** - leaderboards  
  Example: `!lb roulette weekly` or `!lb faction 5`
- **!promo** (`!p`) - current promo  
  Example: `!p`
- **/help** - private help menu  
  Example: `/help`

> Note: If using `!` triggers older bots, then prefix the command with `?` (`?ft` instead of `!ft`)- check `/help` for list.

**Handy Tools**
_Practical helpers for otherwise tedious rpg stuff._

- **/findmyid <name>** - lookup RPG ID given username (or vague recollection of it)  
  Example: `/findmyid name:infinity stones`
- **/viewbox** - DMs your box to you  
  Example: `/viewbox id:123456`
- **/sortbox** - sort your box and DM the BBCode. Customizable options.  
  Example: `/sortbox id:123456`
- **!sprite <pokemon> [xy|hgss|bw]** - sprite URL  
  Example: `!sprite Grookey bw`
- **!wiki <term>** - wiki search  
  Example: `!wiki goldenize`
- **!pokedex <pokemon>** - full dex info  
  Example: `!pokedex Aegislash`
- **!dexname <dex #>** - name lookup by dex number  
  Example: `!dexname 123`
- **!stats <pokemon>** - stats only  
  Example: `!stats Cryogonal`
- **!eggtime <pokemon>** - breeding time  
  Example: `!eggtime Larvesta`
- **!count** - message counts and leaderboard  
  Example: `!count leaderboard`
- **/notifyme set <phrase> [ignore_users]** - DM on phrase with optional ignored users  
  Example: `/notifyme set phrase:golden ignore_users:@user1 @user2`
- **/notifyme ignore <phrase|phrase_id> <users>** - add ignored users to an existing watched phrase (autocomplete enabled)  
  Example: `/notifyme ignore phrase:1 users:@user1 @user2`
- **/remindme set <phrase> <time|at>** - timed reminder (supports today/tomorrow)  
  Example: `/remindme set phrase:trade at:7am tomorrow`
